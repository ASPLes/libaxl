Welcome to PyAxl's documentation!
=================================

PyAxl is a binding to use Axl Library from your python
applications. It provides the same effective XML processing and a
similar API like does Axl Library, but available from python. 

See Axl Library base documentation for more details about Axl Library
features:

  - Axl Library doc center <http://www.aspl.es/xml/html/index.html>
