Acerca de Axl Library

"Axl Library" es una implementación de código abierto del estándar XML 1.0, escrita por ASPL para soportar muchos de sus requisitos de software XML. Ha sido construida con el objetivo de producir una librería de gran calidad y fuertemente probada, con un alto rendimiento en ejecución junto con un consumo de memoria bajo.

Fue inicialmente escrita para ser usada por la plataforma Af-Arch debido a la gran cantidad de datos transferidos. Sin embargo, la librería ha evolucionado para ser utilizada de manera independiente por otros proyectos.

ASPL es una compañia española que proporciona servicios profesionales para introducir en las organizaciones y empresas, sistemas de código abierto, principalmente basados en GNU/Linux.
