Axl library para CentOS 6 / Centos 7

Paquetes para CentOS también están disponibles con el siguiente repositorio yum:

# /etc/yum.repos.d/ASPL-Base.repo
# Aspl-Public repo 
#
# ASPL packages for libaxl, vortex, nopoll, myqtt, ext-dns, valvula
# and other open source public projects
#

[aspl-public]
name=Aspl-Public-$releasever - Base
baseurl=http://www.aspl.es/centos/public/$releasever/$basearch/
gpgcheck=0

# Aspl-Private repo 
#
# ASPL packages for commercial projects like core-admin
#
[aspl-private]
name=Aspl-Private-$releasever - Base
baseurl=http://www.aspl.es/centos/private/$releasever/$basearch/
gpgcheck=0

Ejecute lo siguiente para buscar paquetes de axl: 

>> yum --disablerepo="*" --enablerepo="aspl-public" list available

Github / Subversion

Aunque las secciones anteriores contienen referencias a las revisiones estables, también puede descargarse el último estado de Axl Library.

Si encuentra algo que falte o que no está implementado correctamente, puede enviarnos un correo a la lista de correo de Axl o comprobar el estado actual del SVN/Github.

Compruebe las siguientes urls para obtener un enlace al último código fuente:

Github y SVN para la última revisión estable:
svn co https://github.com/ASPLes/libaxl
git clone https://github.com/ASPLes/libaxl
