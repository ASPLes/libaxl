  • 2006-11-09 LibAxl 0.3.0 está lista, ¿cuanto de rápido es su toolkit XML?

    Esta nueva revisión ha consegido completar uno de los objetivos principales de Axl Library: producir un toolkit XML de alta calidad, con una interfaz limpia y que proporcione un rendimiento elevado en tiempo de ejecucción con un consumo de memoria bajo.

    Usando LibXML como referencia, posiblemente una de las implementaciones XML más rápidas actualmente disponibles, esta nueva revisión de Axl Library muestra que es aun más rápida mientras procesa documentos XML y al mismo tiempo consume cerca de 2.5 veces menos memoria. Vea todos los detalles en el informe de consumo y rendimiento: Axl Library 0.3.0 performance and memory report.

    Puede encontrar todos los detalles de esta revisión en: Axl Library 0.3.0
