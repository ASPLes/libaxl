Axl News

  • 2006-11-09 LibAxl 0.3.0 is ready, how fast is your XML toolkit?

  • This new release makes Axl Library to reach one of its objetives: to produce a high quality xml toolkit, with a clean interface, that provides good performance results at runtime execution and memory consumption.

    Using LibXML as reference, which maybe is one of the fastest xml toolkit available, this Axl release shows to be faster than it while processing xml documents and, at the same time, consuming around 2.5 times less. See the following report to get more details: Axl Library 0.3.0 performance and memory report.

    You can get full details at the release note: Axl Library 0.3.0

  • 2006-11-09 LibAxl 0.2.7 is out: minor bug fixes release
  • A new Axl library release has been reached and it is ready to download. This new release includes bug fixes while handling attributes and a new API to support axlList iteration in a linear way.

    You can get full details at the release note: Axl Library 0.2.7

  • 2006-10-23 LibAxl 0.2.6 is out: more platforms supported release
  • Axl library 0.2.6 is ready to download. This new release fix Axl library to compile and properly work on Amd64 running Linux. Due to this, checked that Axl Library builds and runs properly at: Alpha (running Linux), Openpower (running Linux), PowerPC (running Linux), NetBSD, OpenBSD, FreeBSD, Windows and Linux (all of them running on x86).

    More API to the library has been added and more library performance improvement has been done. See memory report.

    You can get full details at the release note: Axl Library 0.2.6
