Axl News

  • 2006-10-09 LibAxl 0.2.5 is out: the MIXED content release
  • Axl library 0.2.5 is ready to download. This new release includes support to inspect MIXED content (comments, xml nodes, content, process instructions, etc), improves memory usage and library performance (see memory report).

    You can get full details at the release note: Axl Library 0.2.5

  • 2006-07-07 LibAxl 0.2.4 is out!
  • Axl library 0.2.4 is ready to download. This new release improves the DTD validation support and adds more API to interface with XML documents. This release has been used to produce the new xml-rpc-gen tool (an IDL/XDL protocol compiler), inside Vortex Library project.

    Check full details here: this link

  • 2006-05-03 LibAxl 0.2.3: the boolean fix!
  • A new Axl Library release is available. This release solves an issue produced by a boolean definition, that was causing to not properly compile the library under c++ environments. Check full details here: this link
  • 2006-04-28 LibAxl 0.2.2 released!
  • A new release is now available. Check full details on: this link
  • 2006-04-20 LibAxl 0.2.1 released!
  • First official LibAxl release. With this release, the Af-Arch and the Vortex Library project have started to use it. We hope you find it useful.

    You can find full release note detail on: this link
