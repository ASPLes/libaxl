Axl News

  • 2008-11-11 LibAxl 0.5.5, "The place where things makes sense" release is ready!
    Next stable release applying axl-bool-removal-02 change notification to fully fix errors found on previous change notification proposal. This release improves C++ library usability. See all details at: Axl Library 0.5.5: "The place where things makes sense" release note.
  • 2008-10-24 LibAxl 0.5.4, "Friday, I'm in Love!" release is ready!
    Next stable release including bug fixes and an API change defined by the change notification axl-bool-removal-01. See all details at: Axl Library 0.5.4: "Friday, I'm in Love!" release note.
  • 2008-06-26 LibAxl 0.5.3, "Don't touch the PRNG!" release is ready!
    Next stable release including fixes and new functions at the String API. This is a fix release. See all details at: Axl Library 0.5.3: "Don't touch the PRNG!" release note.
  • 2008-05-12 LibAxl 0.5.1, "La torre de babel" release is ready!
    Next major stable release. Base library (libaxl) was updated to allow configuring a set of external handler that allows to perform run-time buffered decode operations. This was used to implement libaxl-babel, the foundation to give Axl Library support for more encoding formats than utf-8.

    Fixed several bugs at the library and some improvements were implemented on axl-knife (like producing a C inline representation for a DTD).

    See all details at: Axl Library 0.5.1: "La torre de babel" release note.
  • 2008-01-24 LibAxl 0.4.14, "Bright lights" release is ready!

  • 2006-11-09 LibAxl 0.3.0 is ready, how fast is your XML toolkit?
  • This new release makes Axl Library to reach one of its objectives: to produce a high quality xml toolkit, with a clean interface, that provides good performance results at runtime execution and memory consumption.
    Using LibXML as reference, which maybe is one of the fastest xml toolkit available, this Axl release shows to be faster than it while processing xml documents and, at the same time, consuming around 2.5 times less. See the following report to get more details: Axl Library 0.3.0 performance and memory report. You can get full details at the release note: Axl Library 0.3.0
