About Axl Library

Axl Library is an open source XML 1.0 implementation written by ASPL to support many of its XML software requirements. It was built with the objective to produce a high quality and well tested library with a good runtime performance and low memory consumption.

It was initially written to be used by the Af-Arch platform, but now is used by many other projects..

ASPL is a Spanish company that provides services to introduce GNU/Linux systems inside organizations.
