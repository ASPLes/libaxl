About Axl Library

Axl Library is an open source XML 1.0 implementation written by Advanced Software Production Line, S.L. to support many of its XML requirements. It was built with the objective to produce a high quality and well tested library with a good runtime performance and low memory consumption.

It was mainly written to be used by the Af-Arch platform, because the amount of data transfered on each request. However, it has evolved to be usable in an stand alone way.

Advanced Software Production Line, S.L. is a Spanish company that provides services to introduce GNU/Linux systems inside organizations. It offers two different services:

  • Software development services, based on Open Source technologies, especially those designed by Advanced Software Production Line, S.L.: Af-Arch and Vortex Library.
  • GNU/Linux servers: providing all services around its installation and its support.
